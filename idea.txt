Идея разработки MewAI
MewAI это веб платформа для генерации ответов от ИИ ввиде текста а так же ввиде кода, данная платформа будет полностью бесплатной и будет иметь специальный функционал которого нету у других.

Страницы:
index.html - первая страница которая создана для входа или регистрации в системе \\\ вход осуществляется при помощи никнейма и пароля которые сохранены в базе данных Supabase \\\ а так же при регистрации пользовать указывает свой никнейм для входа а так же пароль и пишет его еще раз для подтверждения и в итоге пишет свой никнейм телеграм или электроную почту для обратной связи. После входа сохраняется в local storage его данные для входа и когда пользователь заходит на эту страницу то его сразу переносит на страницу генерации логинясь автоматически.
html/main.py - главная страница на которой есть весь контент, она содержит в себе показ страницы chat.html а так же некоторые элементы которых нету у каждого ИИ.
    1. Справа вверху есть 3 полоски [../icons/menu.png] которые открывают меню слево которое имеет такие элементы как список чатов которые сохранены в Supabase а так же ниже кнопки которые переносят на другие страницы \\\ профиль -> profile.html \\\ статистика -> stats.html \\\ поддержка -> donate.html
    2. Cлево сверху есть иконка [../icons/ghost.png] которая переносит на ghost.html
    3. Показывает на весь экран chat.html и затемняет при открытии 1 элемента
html/chat.html - сам чат с ботов в котором есть несколько важных элементов:
    Элементы:
        1. Cтрока ввода запроса в центре экрана есть чат новый и над строкой написанно время пользователя
        2. У строки ввода тоже есть несколько элементов которые очень важны
            1. Справо строки ввода есть кнопка [../icons/files.png] которое позволяет привязывать файл к строке ввода
            2. Слево строки ввод есть выбор ответа такие как fast[../icons/fast.png] и thinking[../icons/thinking.png] который влияет на промпт и ответ ИИ в бекэнде
            3. В самом левом углу есть кнопка для отправки [../icons/sent.png] но так же отправить можно при помощи кнопки enter
        3. В самом низу стоит место для баннера 300 на 200 которое не мешается для ввода но при этом же там находится и пусть для него будет специальное место чтобы оно не ломало интерфейс
html/profile.png - на данной странице показано использования пользователем бота и статистика использования \\\ на сайте показаны такие данные как \\\ всего потраченно токенов для генерации, имя и пароль а так же сменить пароль (пароль засекречен и не показывается пока ты на него не нажмешь чтобы было безопасно )
html/stats.png - тут показывается сколько всего было потраченно токенов для каждого из API токенов для генерации и их нагруженность ( максимальная выдержка токена это 1000000 токенов за 7 дней )
html/donate.html - Страница с донатами на которой есть реквизиты PayPal а так же номер моей лички чтобы стать донором API токена для проекта и получать часть дохода с рекламы
html/ghost.html - точно такая страница как и chat.html но которая хранит данные в local storage и они удаляются при перезагрузки или выхода с страницы а так же страница отличается стилем

Интерфейс:
    1. Цвет фона: #211D2D \\\ цвет текста: #F2DFD8
    2. Цвет фона для ghost.html: #363A45 \\\ цвет текста для ghost.html: #667C70

Бекэнд:
py/main.py - обьядиняет весь функционал всех файлов в папке
py/generation.py - генерирует файл при помощи кода который указан ниже а так же использует разные 25 токенов в .env и использует их по кругу не нагружая один токен сильно 
py/database.py - база данных Supabase и все что нужно для сохранения данных

Dotenv файл:
SUPABASE_URL=
SUPABASE_KEY=
25 токенов - просто токены для генерации ответа которые крутятся по кругу в generation.py чтобы не воткнуться в лимиты и тп

Код для генерации ответа:
import os
from ollama import Client

client = Client(
    host="https://ollama.com",
    headers={'Authorization': 'Bearer ' + os.environ.get('OLLAMA_API_KEY')}
)

messages = [
  {
    'role': 'user',
    'content': 'Why is the sky blue?',
  },
]

for part in client.chat('gemma4:31b-cloud', messages=messages, stream=True):
  print(part['message']['content'], end='', flush=True)

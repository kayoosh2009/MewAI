# MewAI Project - by Kayoosh Studio
Mew AI это телеграм бот который работает на базе моделии Gemma 4 от гугл и позволяет бесплатно и почти без лимитов пользоваться моделью в любом уголке страны.
Бот доступен через телеграм бота @ne1ronekobot

Функционал бота:
 - Отвечает текстом на любом языке
 - Анализ фото
 - Система баллов генерации

Бэкенд функционал:
 - Система токенов, бот работает на 10 ключах с разных аккаунтов для максимального удобства и предотвращения овер лоудинга
 - Хранение даннных на SupaBase
 - Система памяти и очистки данных

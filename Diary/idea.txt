Идея разработки:
MewAI это телеграм бот с встроенным ИИ от Ollama Cloud на бесплатной основе

Файлы:
  main.py - главный код написанный на пайтоне который выполняет все что нужно ( команды, расчеты, ответы и тп )
  language.json - разные виды ответов на русском и англйиском а так же системные промпты для ИИ
  data/users.json - Голый список пользователей ( никней / id / дата входа в бота )
  data/chats.json - список чатов каждого пользователя
  data/queque.jsonl - список ожидания
  data/stats.json - статистика каждого пользователя
  data/earn.json - Ссылки и тп

Команды у бота:
  /start - присылает список команд на английском и так же строчку на русском с командой по смене языка
  /stats - присылает статистику пользователя ( имя / айди / балланс )
  /clear - очистить историю чата в chats.json а так же полностью стирает чат до команды /start 
  /earn - заработать монетки
  /lang - поменять язык ( русский / английский )

Способы заработка монет:
  1. Посмотреть пост телеграм - 10 монет
  2. Отреагировать на пост в телеграм - 20 монет
  3. Подписаться на канал - 100 монет
  4. Ежедневный вход - 15 монет

Цена монеты к токенам и алгоритм
  1 Монета = 5300 токенов 
  При генерации ответов ИИ может написать больше чем есть токенов у пользователя и тогда в балансе может быть минус но если у пользователя минус то ему нужно будет выполнять задания.
